# Agent Capabilities

Central reference for what each agent can do. Used by all agents for routing and capability discovery.

---

## Larry — Orchestrates (Text & Code)

**Modality:** Text, code, planning, memory
**Model:** Claude (Sonnet/Opus/Haiku depending on task)

### Capabilities
| Capability | Tool / Method | Notes |
|------------|---------------|-------|
| Write, edit, analyze text | Claude native | Multilingual |
| Code, debug, refactor | Claude + Bash | All languages |
| Web search | WebSearch + WebFetch | Live data |
| Browser / UI automation | Playwright (MCP) | Persistent Edge profile |
| Gmail (read, send, archive, label) | GWS CLI (`gws`) | Never MCP |
| Google Calendar (read, create events) | GWS CLI | |
| Google Drive (read files) | GWS CLI | |
| Semantic search in vault | Milla / MemPalace MCP | 19 tools |
| Knowledge graph (KG) | Milla kg_add/query | Entity-relation-object triples |
| Diary memory | Milla diary_read/write | AAAK format |
| Traverse Milla's palace | Milla traverse/find_tunnels | Wings, rooms |
| Create and edit vault files | Write + Edit | Always vault-first |
| Git (commit, push, branch) | Bash + git | |
| Run Python scripts | Bash | Barry/Harry agents |
| Task Scheduler (Windows) | Bash + PowerShell | Nightly automation |

### Skills

Skills files define Larry's approach per domain — which frameworks activate, how user context is applied. Loaded on demand, indexed by Milla.

Active skill domains: brand-thinking, business-law, seo-sem, business-planning, strategy-thinking, leadership, coaching, marketing, sales-thinking, communication, project-management, financial-reasoning.

### GWS Apps (Google Workspace CLI)
- **Gmail** — read inbox, send, archive, label
- **Calendar** — fetch events, create meetings
- **Drive** — read and search files
- **Sheets / Docs / Tasks** — planned, not yet implemented

## Barry — Sees (Image & Visual Intelligence)

**Modality:** Image — generation, analysis, organization
**Platform:** Venice.ai (Playwright) + FFmpeg

### Capabilities
| Capability | Tool / Method | Notes |
|------------|---------------|-------|
| Generate images (SFW) | Venice Chat via Playwright | Free model (Chroma) |
| Generate images (NSFW) | Venice Chat via Playwright | Various models |
| Upscale images (2x / 4x) | Venice Upscale via Playwright | creativity 0.20, adherence 0.70 |
| Analyze / interpret images (vision) | Venice vision via Playwright | QA, categorization, tagging |
| Sort and categorize images | barry-sort.py | vision -> folder structure |
| Convert HEIC -> JPG/PNG | Python / ffmpeg | NAS inbox flow |
| Name images | barry-NNNNN.ext (global counter) | Sequential naming |
| NAS backup | Robocopy -> NAS mount | After EVERY operation |
| Build visual index | Milla mine + visual-index/ | Search among images |

### Image Catalog
- **generated/** — everything generated by Barry
- **import/** — imported (photos, screenshots)
- **Subfolders:** nsfw/ sfw/ -> category -> filename
- **Trash:** `{{ASSETS_PATH}}/.trash/` — 30-day retention

## Parry — Guards (Privacy & Tone)

**Modality:** Filter and judgment layer — sits between Larry and the outside world
**Model:** Claude (fast Haiku for hooks, Sonnet for deep review)

### Capabilities
| Capability | Tool / Method | Notes |
|------------|---------------|-------|
| Privacy enforcement (L1-L4) | parry.py | Blocks violations BEFORE they happen |
| Pre-commit hook (privacy scan) | .git/hooks/pre-commit | Scans staged changes |
| Pre-mail hook | Reviews attachments + body | Before GWS send |
| Wikilink review | Parses [[links]] | Public -> private = block |
| Tone adjustment | Context-aware | Different tone per channel |
| Privacy audit of vault | parry.py audit | Manual / nightly |
| Semantic privacy scan | Milla search | Finds leaked private content |

### Parry Tone Contexts
| Recipient | Tone |
|-----------|------|
| LinkedIn | Professional, thought leader |
| Reddit | Casual, community |
| Clients | Consultant, clear |
| Close contacts | Personal, vulnerable |
| Friends | Direct, no filter |
| Vault (internal) | Raw, unfiltered |

## Ecosystem Flow

```
User
  |
  v
Larry (orchestrator)
  |-- Milla (memory + search)
  |-- Barry (image) --> Venice via Playwright
  |-- Harry (audio) --> Gemini TTS/STT/Live + Venice music/SFX
  |-- Parry (gatekeeper) --> privacy + tone
  +-- GWS (communication) --> Gmail + Calendar + Drive
```
